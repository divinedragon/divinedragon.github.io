[taskwarrior-tui][1] - A Terminal User Interface (TUI) for [Taskwarrior][3] that you didn't know you wanted.

![Task Warrior Terminal User Interface Demo][2]

### Features

* vim-like navigation
* live filter updates
* add, delete, complete, log tasks
* multiple selection
* tab completion
* colors based on [taskwarrior][3]
